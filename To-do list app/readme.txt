# To-Do List Application

A simple to-do list application with a graphical user interface (GUI) created using Tkinter in Python.

## How to Run the Application

1. Clone the Repository:

    ```bash
    git clone https://github.com/ishant06/todo-list-app
    ```

2. Navigate to the Project Directory:

    ```bash
    cd todo-list-app
    ```

3. Run the Application:

    ```bash
    python todo.py
    ```

    Make sure you have Python installed on your system.

## How to Use the Application

The application provides a basic to-do list management system with the following features:

### Adding a Task

1. Enter the task in the "Enter Task" entry field.
2. Click the "Add Task" button.

### Deleting a Task

1. Enter the index of the task you want to delete in the "Enter Task" entry field.
2. Click the "Delete Task" button.

### Marking a Task as Completed

1. Enter the index of the task you want to mark as completed in the "Enter Task" entry field.
2. Click the "Mark Completed" button.

### Viewing the Task List

- Click the "Show Tasks" button to view the list of tasks in the application window.

## Notes

- The application has a light blue background.
- Buttons have curved borders with a white border color.
- The task list is displayed directly within the application window.

Feel free to customize and extend the application based on your requirements!
